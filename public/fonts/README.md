# Font Files

## Spumoni LP W01 Regular

This directory should contain the Spumoni LP font files:

- `SpumoniLP-Regular.woff2`
- `SpumoniLP-Regular.woff`

### Where to Get Spumoni LP

Spumoni LP is a commercial font. You can obtain it from:

1. **Fonts.com (Monotype)**: https://www.fonts.com/font/linotype/spumoni
2. **MyFonts**: https://www.myfonts.com/collections/spumoni-font-linotype
3. **Adobe Fonts**: May be available if you have an Adobe Creative Cloud subscription

### File Format

After purchasing, you'll need to convert the font to web formats (WOFF and WOFF2) if not already provided:

- Use online converters like Font Squirrel's Webfont Generator
- Or use FontForge/similar tools to convert TTF/OTF to WOFF/WOFF2

### Installation

1. Place the font files in this directory (`public/fonts/`)
2. Make sure the filenames match:
   - `SpumoniLP-Regular.woff2`
   - `SpumoniLP-Regular.woff`
3. The font is already configured in `src/index.css` and will load automatically

### Alternative

If you cannot obtain Spumoni LP, you can use a similar font by editing `src/index.css` to point to a different font file.
